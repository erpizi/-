# SpotifyFloatLyrics

这是一个可导入 Android Studio 的 Kotlin 工程，用来实现：
- 监听 Spotify 通知识别当前歌曲
- 从 LRCLIB 搜索歌词
- 用系统悬浮窗显示歌词
- 支持拖动悬浮歌词位置

## 你要做的事
1. 用 Android Studio 打开这个项目
2. 等 Gradle 同步完成
3. 连接手机后直接运行，或者 Build APK
4. 手机里手动开启：
   - 悬浮窗权限
   - 通知读取权限
   - 后台运行/自启动（部分国产机需要）

## 注意
- 这是最小可用版，不是最终商业版
- 同步歌词目前按“歌词开始时间”本地推进，不是真正读取 Spotify 精确播放进度
- 冷门歌曲可能搜不到歌词
- 如果你要我继续升级，可以再做：
  - 字体大小/透明度设置
  - 锁定位置
  - 点击穿透
  - 真正按播放器进度同步
  - 本地缓存
  - 仅在 Spotify 前台显示
